# Contributing to Oracle Fusion AI Agent Studio Sample Apps and Workflows

## Opening issues

Please let us know your ideas, missing features, or bugs found. Please [file a RFE/bug](https://github.com/oracle/fusion-ai-studio/issues/new/choose)

Any information, ideas or suggestions that you provide to us either through a Bug Report or a Feature Request ([https://github.com/oracle/fusion-ai-studio/issues/new/choose](https://github.com/oracle/fusion-ai-studio/issues/new/choose)) shall be considered Feedback. If you think you've found a security
vulnerability, do not raise a GitHub issue and follow the instructions in our [security policy](./SECURITY.md).

## Contributing code

For legal reasons, we cannot accept external pull requests. 

## Pull request process

**Pull requests are currently not being accepted for the Oracle Fusion AI Agent Studio Sample Apps and Workflows project.**
